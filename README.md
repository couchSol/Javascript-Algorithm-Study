## 진행방식
* 매주 정해진 범위를 공부합니다
* 주 1회 해당 범위의 문제를 페어로 풉니다
* 공부한 내용을 블로깅 or 코드 풀이를 적습니다

## 레퍼지토리 활용

* full request로 업데이트합니다
* 마크다운도 연습하고 잔디도 심습니다
